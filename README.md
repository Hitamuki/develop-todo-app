# ToDoアプリケーション

すべての工程・すべての技術領域を、実務を想定して一人で担当する。

- ベンダー
  - 企画、市場調査、競合分析、設計構想、RFP、要求定義、仕様変更、マーケティング、ヘルプデスク
- SIer
  - 要件定義〜運用・保守
  - フロントエンド、バックエンド、インフラ（クラウド）
